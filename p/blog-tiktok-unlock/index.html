写在前面#

众所周知，TikTok 会锁区，在国内是无法正常使用的，好早前就想上 TikTok 耍耍，一直也没成功。

今天逛 GitHub 的时候看到这个方法，但是项目 README 写的很不全，所以笔者就写了这篇文章来记录一下。

准备工作#

首先需要准备一个美区 Apple ID，以及一个科学上网软件 Shadowrocket / Quantumult X / Surge 等。

另外需要安装的软件有：

  • iTunes v12.6.5.3（需要低版本，64 位 | 32 位）
  • 爱思助手（同时需要 USB 线连接手机）
  • iOS 任意版本号APP下载 v6.0：52pojie 帖

配置代理#

基本可以 Follow 这个 GitHub 的仓库 README，不过他写的有点乱，笔者这里还是稍微整理一下，因为笔者只有 Shadowrocket，所以只能以 Shadowrocket 为例。

info

注意如果你之前安装过 TikTok 请卸载再配置代理！！！

  1. 打开 Shadowrocket

  2. 点击下方的 配置，点击自己使用的配置后面的 i -> HTTPS 解密 -> 开启 HTTPS 解密 -> 生成新的CA证书 -> 安装证书

  4. 打开手机设置 -> 通用 -> 关于本机 -> 证书信任设置 -> 找到 Shadowrocket，开启信任

下载旧版本 TikTok#

首先下载和安装 iTunes，在 iTunes 登录美区账号。

打开 iOS 任意版本号APP下载 v6.0，选择美国，搜索 TikTok，右键查看历史版本。

往下滑找到 21.1.0 版本，右键下载。

会进入拦截页面，此时再在 iTunes 搜索 TikTok，找到对应的 APP，点击下载。

这时候可能会下载不成功，但软件已经拦截成功了，所以可以暂停下载，停止拦截，再重新开始下载即可。

iTunes 下载的默认路径在 C:\Users\用户名\Music\iTunes\iTunes Media\Mobile Applications，找到下载好的 TikTok 21.1.0.ipa 文件。

安装 TikTok#

打开爱思助手，将手机连接到电脑，并点击 信任。

如果之前没用过，会在手机上安装 爱思助手极速版，并需要打开一下，不然可能会因为服务未启动而无法安装。

点击 应用游戏，从本地文件导入进行安装，找到刚刚下载的 TikTok 21.1.0.ipa 文件安装即可。

如果打开让你选择感兴趣的话题，就说明安装成功了！恭喜你可以正常使用 TikTok 了！
